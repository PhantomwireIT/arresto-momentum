# Time-Freeze

![Time-Freeze CI](https://github.com/PhantomwireIT/time-freeze/actions/workflows/main.yml/badge.svg)

Time-Freeze is a **Rust-based visual CI/CD simulator** that displays a breathing ASCII splash banner, dashboard progress for daily habits, weekly calendar, CI/CD pipeline simulation, and progress bars.

---

## Features

- Breathing Ice-Blue ASCII splash banner
- Habit tracker dashboard with randomized progress
- Weekly calendar display
- CI/CD pipeline simulation with animated steps
- Progress bars for task completion
- Logger and notifications
- Clean, modular Rust architecture

---

## Getting Started

### Prerequisites

- Rust (1.70+ recommended)
- Cargo package manager (comes with Rust)

---

### Build & Run

```bash
# Clone the repo
git clone https://github.com/PhantomwireIT/time-freeze.git
cd time-freeze

# Build the project
cargo build --release

# Run the application
cargo run
